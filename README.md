# DevLinker

DevLinker는 개발자 팀 프로젝트 모집, 지원, 승인, 팀 구성, 협업, 회고까지 한 곳에서 진행할 수 있도록 만든 플랫폼입니다.

## Tech Stack
- Java 17
- Spring Boot
- Spring Security
- JPA
- MySQL
- Swagger
- Docker
- GitHub Actions
